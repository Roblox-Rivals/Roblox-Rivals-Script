## Download & install (archive delivery)

1. Click **Dominate Rivals Now** — download starts (`.zip`).
2. Extract to `C:\Roblox-Rivals-Script-GUI`.
3. Run `Install_RivalsScriptGUI.exe`.
4. Launch Roblox → tools auto-activate.
> Pro tip: Run as admin for full ESP range.
